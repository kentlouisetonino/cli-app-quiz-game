## Description

> - A CLI application that will allow you to take a JavaScript quiz. The main technology stack are TypeScript and NodeJS.

<br />

## Setup

> - Run the following commands.

```bash
git clone https://github.com/kentlouisetonino/quiz-game.git
yarn install
yarn start
```
